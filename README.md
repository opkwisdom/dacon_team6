## 📌 Stage 설명

#### Stage1 데이터 불러오기

- 데이터불러오기
- 데이터 살펴보기

#### Stage2 데이터 확인

- 데이터프레임 정보확인
- 결측치 처리하기
- 전처리 데이터 저장

#### Stage3 모델링 기초

- 모델링
- 레이블 인코딩
- 데이터 나누기
- 모델 학습 및 평가
- 예측 및 파일제출

#### Stage4 EDA

- 종속변수 확인
- 수치형 데이터 EDA
- 카테고리형 데이터 EDA
- 데이터 시각화

#### Stage5 Feature Engineering

- 불필요한 변수 제거
- 치우쳐진 데이터(Skewed Data) 처리
- 이상치 데이터 (범주형 변수 추가)
- 중복 데이터 처리
- 파생변수 만들기
- 카테고리형 변수 반환

#### Stage6 하이퍼 파라미터 튜닝 & 최종 모델링

- 앙상블 모델
- 의사결정나무/랜덤포레스트/XGBoost/LightGBM
- 모델 성능비교
- 하이퍼 파리미터 튜닝 비교
- 최종 하이퍼 파라미터 적용
- 예측 결과 저장 및 제출
